重装系统前的准备

发表于 2023-04-14

经历了人生中第一次（被迫）重装系统，有些数据的恢复花了好久才探索出来，想总结一下这几天的痛苦经历，重装系统或者购入新机的朋友可以看看。

@TOP

0. 桌面恢复

之前 C 盘满了，我把系统默认的桌面、视频、文档等文件夹都放到了 D 盘，重装系统之后，D 盘中还存着桌面等数据，但是桌面空无一物。

解决方案：移动桌面等文件夹的存储位置，具体操作如下

  • 在桌面图标上右键，选择“属性”–“位置”–“移动”选择待存储位置。
  • 其他文件夹同理。
  • 之后桌面将会显示之前的桌面内容。

1. 网络

1.1 网络连接

C 盘重装系统之后，再次连接网络需要重新输入网络密码。
解决方案： 提前记录密码，做好准备，避免忘记。

1.2 VPN 连接

VPN 的安装我放在 D 盘，重装之后不影响使用，但是需要重新下载某些网络组件，在启动小飞机的时候会显示，选择安装并耐心等待即可。

2. 下载相关软件

2.1 google 浏览器信息恢复

本人是谷歌浏览器的重度依赖患者，而谷歌浏览器安装又是默认安装在 C 盘的，所以理所当然的，我的浏览器没了。需要在新系统上重新安装。
解决方案：

  • 联网问题和 VPN 问题解决；
  • 在微软自带的 edge 浏览器上搜索 google 浏览器，下载网址在此。下载安装包之后点击即可自动完成安装。
  • VPN 没问题之后打开谷歌浏览器，登录谷歌账号
  • 建议之前开启了同步功能，一瞬间自己的书签收藏夹、界面设置都会恢复原样。

2.2 Microsoft 相关软件安装

重装系统貌似只负责装系统，word、excel、OneNote 之类的刚需软件都搜不到。

解决方案：在官网下载安装包，下载网址在此，安装即可

2.2.1 Onenote 数据恢复

Onenote 是我在电脑端常用的笔记记录方式，大概存了我两年的笔记，突然重装系统让我失去了原本存储在 C 盘的临时备份文件。

解决方案

  • 重装系统之前的操作：onenote 中选择“文件”–“选项”–“同步”–勾选“自动同步笔记本”选项
  • 重装系统之后，在官网下载 Onenote 安装包，下载网址在此
  • 下载完成之后，此时的 Onenote 登录自己的 microsoft 账号，但是没有任何笔记内容。
  • 【重点！】，进入网页版onedrive 网站,使用自己的账户登录之后在“文档”中查找，可找到自己的笔记文件。点击想要打开的文件后的“···”，选择“打开”–“在应用中打开”。

2.3 QQ 与微信【修改文件下载的默认路径】

网络问题解决之后，只需要在官网下载 QQ 和微信安装包即可。
注意

  1. 安装软件尽量安装在 D 盘，避免后面 C 盘爆满的时候不好清理。

  2. 安装完成之后，QQ 和微信文件下载的默认路径是 C 盘，后续在使用过程中将会导致 C 盘存储空间越来越小，影响电脑使用，因此在安装新软件之后最好修改一下文件的下载路径：

    1. QQ 的文件下载路径修改
      在 D 盘新建一个文件夹，打开 QQ 电脑端，点击左下角的“主菜单”–“设置”–“文件管理”–“更改目录”–选择 D 盘刚刚新建的文件夹。

    2. 微信的文件下载路径修改

      • 在 D 盘新建一个文件夹，打开微信电脑端，点击左下角的“主菜单”标识–“设置”–“文件管理”–“更改”–选择 D 盘刚刚新建的文件夹。

      • 如果自己加入了很多群聊，自动下载可能会导致群聊里面和自己无关的图片、文件被大量下载，可考虑关掉这个功能，后续手动选择自己需要的文件进行下载即可。

3. vscode 设置

vscode 我安装在 D 盘，但是 C 盘重装系统之后，,vscode 还是没办法启动，我猜测可能是注册表的问题。所以选择重新安装，下载网址在此。

3.1 插件安装位置

相信大家很喜欢 VScode 强大的插件包容性，各种插件简直就是编码写文的利器，但是，这些插件可是实实在在下载到电脑里面吃内存的，VScode 默认的插件安装位置是 C 盘，例如我的电脑里面，插件安装位置就是：C:\Users\25217\.vscode\extensions。
这样的话，随着插件的使用越来越多，最终将会导致 C 盘宝贵的内存被占。

4. zotero

zotero 主要是用于管理论文，当论文导入的时候，论文的副本和注释均放在C\Users\用户名\Zotero，可修改论文存储位置。
在其他盘新建文件夹，将原 C 盘 zotero 文件夹的论文复制到新文件夹中。
在 zotero 上方工具栏中选择：编辑-首选项-文件和文件夹，然后选择自定义文件夹。

5. 重装系统的经验总结

  1. 维修电脑前重要信息做好备份，例如插件、论文资料、桌面文档、网络密码等，避免黑心维修人员临时说需要重装系统；
  2. 软件尽量安装在 D 盘，重装系统之后有的软件没有注册表可能会失效，再次安装即可；
  3. 常用软件的默认安装位置都是 C 盘，需要人为修改为其他盘；
  4. 桌面等系统自带文件夹也可以修改为 D 盘存储，能够直接还原原始桌面；
  5. 浏览器等软件开启备份同步功能做好预备。
